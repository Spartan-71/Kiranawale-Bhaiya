> *Your smart bhaiya who knows you’ll run out of atta before you do.*

## Why this exists

Most grocery apps designed for you to decide what to order.

**Kiranawala Bhaiya tells you what you *need* beforehand.**

---

## ⚙️ How it works

This runs quietly in the background. Every day it:

1. Pulls your Instamart order history
2. Learns how fast you consume items
3. Predicts what’s about to run out
4. Suggests a ready-to-order cart
5. Executes checkout when you confirm

---

## 🧠 Core Intelligence (in short)

This doesn’t use heavy ML. It uses simple but powerful patterns from your order history:

- Tracks **when you buy each item** → learns consumption rate
- Calculates **average gap between purchases** → predicts next depletion
- Uses **frequency** → prioritizes essentials over rare items
- Learns **items bought together** → smart combo suggestions
- Remembers **preferred variants** → same brand/size auto-selected

This creates a simple system that feels like:

> "Bhaiya knows what I need before I do"

## ✨ Features

### Core Intelligence

- **Consumption modeling** — learns how fast you use each item based on past orders
- **Proactive restocking** — nudges you before items run out
- **Variant intelligence** — auto-selects your preferred brand, size, and type

### Shopping Experience

- **Smart cart** — pre-filled cart with items you’re likely to need
- **Budget-aware shopping** — suggests cheaper alternatives or optimized carts to stay within your monthly budget

### Interface

- **Personality** — conversational tone that feels like chatting with a local bhaiya
- **WhatsApp / Telegram native** — no new app, works directly where you already chat

### Multimodal Input

- **Voice support** — add items using natural speech in multiple Indian languages
- **Photo-to-cart** — send product images or lists to automatically add items

### Context-aware Use Cases

- **Recipe mode** — detects ingredients when you mention a dish and adds them to cart
